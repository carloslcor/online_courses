Avaliação Java Script

1) Crie um novo arquivo HTML e faça um script para calcular a média aritimetica de 3 notas, na exibição na página deve-se apresentar as 3 notas e informar qual a média.

2) Faça um script para saber a idade de uma pessoa, através do ano atual e ano de nascimento, na pagina em HTML deve ser mostrado o nome de uma pessoa e sua idade.

3) Elaborar um programa no qual o sejam informados: o valor da compra e o valor pago. O programa deve calcular e apresentar o troco. Se o valor pago não for suficiente, deve-se emitir um aviso.

4) Elaborar um programa no qual seja informado o valor de um produto em dólar e o valor da cotação do dia. O programa deve converter e apresentar o valor em reais.

5) Faça um programa, com uma função que informado um número seja apresentado o caractere ‘P’, se o número for positivo, e ‘N’, se o número for zero ou negativo.
